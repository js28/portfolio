MIT App Inventor Project

September 18, 2018 - October 11, 2018
Computer Science Principles

The objective of this project was to create a functioning app/game for the andriod interface. My partner and I decided to go with one of the advanced games (from the choices given) to program, and after that we went to work. We had to learn and use time management skills in order to code the game in the time slot alloted to coding the base game. Then, when we had finished coding the game and getting feedback, we decided to make some changes and add stuff. After brainstorming, we decided to add more spikes/obstacles to the game, slong with a background to add more color. Working on this project taught us two things; how to manage our time and how to set compromises. When working on this game, my parnter and I had different visions for what the game would look like, but over time we compromised and wound up creating a game we both are proud of. We also had to learn time management. We had to find out how to allocate our time and learn how to break down our work into segments that we could complete, yet still completing the work to meet deadline. This project was overall really fun, and a link to our Project Notebook Documentation is here.

Python Hangman Project

January 23, 2019 - February 5, 2019
Computer Science Principles

For this project my partner, Jake Cooney, and I were given a brief timeframe and told to use our python programming knowledge to program a functioning hangman game. This project taught us both how to blend ideas together and compromise what additions should be made to the game. To add a twist to the game, we added a limit of guessable letters, and changed the game from guessing random words to selecting and guessing phrases. To make the game more cohesive, we added a theme of retro video games to the game. This made our project special and specific to both of our interests. This was my favorite project to work on this year, and here is the link to our documentation so you can look at our work.
